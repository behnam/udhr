Translations

The column “Key” is the internal key used in this database to identify the translations. It has no meaning or relation to any system of tags.

The column “Name” links to the Ethnologue entry for the language. In this column, we use the primary language name given by the Ethnologue, may be followed by a qualifier in parenthesis. You may want to consult the Ethnologue to determine the primary language name if you have difficulty finding a translation by language name.

The column “BCP47” is a tentative assignment of a BCP 47 language identifier. In some cases, there are multiple translations with the same BCP 47 identifier.

The column “OHCHR” links to the OHCHR version.

The column “Stage” indicates our degree of progress as follows:

  • 1: no known complete translation (3)
  • 2: no conversion to Unicode started (103)
  • 3: partially converted to Unicode (often the first article only (1)
  • 4: fully converted to Unicode, to be reviewed (451)
  • 5: complete and reviewed (6)

The column “Deliverables” links to the deliverables of the project:

  • sh: status and history
  • X: XML
  • C: character count
  • T: plain text
  • H: HTML
  • N: notes
